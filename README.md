## ✨ Features

- 🔍 **Device Discovery** – Automatically find cameras on the local network.
- 📡 **Remote Access** – Test common vulnerabilities (with permission).
- 🖼️ **Snapshot Capture** – Take test shots to verify access.
- 📁 **Log Export** – Save results for analysis.
- 🧩 **Modular Design** – Easy to extend with custom modules.
- 🌐 **Cross-Platform** – Works on Windows, Linux, and Android (Termux).
